To run this app:

1. Install python if not already
2. In python, change working directory to the folder with the app
3. Create a virtual environment (venv) in this folder using your terminal: python -m venv .venv
.\.venv\Scripts\Activate
4. Run: pip install -r requirements.txt in your terminal
5. Run: shiny run flex_app.py

Useful tips:
- Ctrl+C in terminal to close app
